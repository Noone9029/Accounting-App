# Paid Saudi-First SaaS v1 Gap Matrix

Planning date: 2026-05-22

This matrix tracks what remains before LedgerByte can move from controlled beta/user-testing to paid Saudi-first SaaS v1. Vercel is the current beta/user-testing deployment surface only, not final production hosting.
